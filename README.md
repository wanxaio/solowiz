# solowiz
OpenVZ Virtualization Control Panel




INSTALLATION
==========================================
execute following code using root privileges

wget --cache=off http://wasiflaeeq.com/solowiz/install.sh

chmod 0755 install.sh

yum -y install dos2unix

dos2unix -o install.sh

./install.sh
